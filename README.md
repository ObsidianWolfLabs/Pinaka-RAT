# 🏹 Pinaka - Advanced Packet Analyzer

## Project Overview

Pinaka is a sophisticated network packet analyzer inspired by Lord Shiva's divine bow from Hindu mythology. Just as Pinaka was a weapon of precision and power, this tool provides precise network analysis with advanced AI-powered capabilities that go beyond traditional packet sniffers.

## 🚀 Installation & Setup

### Prerequisites
- **Operating System**: Windows 10/11 (64-bit)
- **Python**: 3.7 or higher (for development)
- **Administrator Privileges**: Required for raw packet capture
- **Npcap**: Recommended for enhanced Windows packet capture

### Quick Start (Executable)
1. Download the latest release from the distribution package
2. Extract all files to your desired location
3. Install Npcap from [https://nmap.org/npcap/](https://nmap.org/npcap/)
4. Right-click `Pinaka.exe` and select "Run as administrator"
5. Select your network interface and start capturing!

### Development Setup
```bash
# Clone the repository
git clone <repository-url>
cd PacketSnifferX

# Install dependencies
pip install scapy pyinstaller requests ipwhois

# Run the application
python advanced_gui_sniffer.py

# Build executable
python build_executable.py
```

## 📖 Usage Guide

### Basic Operations
1. **Start Capture**: Click the "▶ Start Capture" button
2. **Apply Filters**: Use the filter bar for specific traffic types
3. **Analyze Packets**: Click on any packet for detailed analysis
4. **AI Insights**: Monitor the AI panel for real-time threat detection
5. **Export Data**: Save captures in PCAP format for further analysis

### Advanced Features
- **Custom Filters**: Use Berkeley Packet Filter syntax
- **Threat Analysis**: Review AI-generated security reports
- **Geo-Location**: Track packet origins on the world map
- **Protocol Analysis**: Deep dive into packet structure
- **Statistical Reports**: Generate comprehensive network analysis

### Keyboard Shortcuts
- `Ctrl+S`: Save current capture
- `Ctrl+O`: Open PCAP file
- `Ctrl+F`: Focus filter bar
- `F5`: Refresh capture
- `Esc`: Stop current capture
